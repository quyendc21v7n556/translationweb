# 🌸 CTU Multi-Language Translator 🌸

### Giới thiệu dự án

Đây là dự án **Niên luận** ngành Công nghệ thông tin tại **Đại học Cần Thơ**. Ứng dụng cung cấp giải pháp dịch thuật đa ngôn ngữ thông minh, hỗ trợ người dùng giao tiếp và học tập hiệu quả với giao diện hiện đại và thân thiện.

---

### ✨ Tính năng nổi bật

- **Dịch đa ngôn ngữ:** Hỗ trợ dịch giữa hàng chục ngôn ngữ phổ biến (Anh, Việt, Indonesia, Nhật, Hàn, Pháp, Thái...).
- **Tự động phát hiện:** Tự nhận diện ngôn ngữ đầu vào (Auto-detect).
- **Phát âm kết quả (TTS):** Tích hợp công nghệ Text-to-Speech để đọc bản dịch với giọng đọc chuẩn theo từng quốc gia.
- **Lưu trữ lịch sử:** Tự động lưu lại các bản dịch gần đây vào cơ sở dữ liệu để người dùng tiện tra cứu lại.
- **Giao diện hiện đại:** Thiết kế Dark Mode kết hợp hiệu ứng hoa anh đào, tối ưu trải nghiệm người dùng.

### 🛠 Công nghệ sử dụng

- **Backend:** Node.js, Express.
- **Database:** MongoDB (Sử dụng Mongoose ODM).
- **API Dịch:** MyMemory Translated API.
- **Giọng đọc:** Web Speech API.
